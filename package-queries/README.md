# Search Extensions

This package contains a bunch of queries examples.

## Prerequisites

The Search Extensions package will work with Unity 6000.0 and any other more recent versions.

## Getting Started

More info on search extensions can be found [here](https://github.com/Unity-Technologies/com.unity.search.extensions/wiki).


This [article](https://github.com/Unity-Technologies/com.unity.search.extensions/wiki/Collection-Tool) explains how to build Live Collection and Object Explorers using the Search Window and SearchQueries.
